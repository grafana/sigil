# Get started with Grafana Sigil

{{< docs/public-preview product="Grafana Sigil" >}}

Get started by instrumenting your agents with a Sigil SDK and deploying Sigil to receive generation data.

## Instrument your agents

Choose the SDK for your language. Each guide walks you through installing the SDK, capturing your first generation, and verifying data in Sigil.

- [Python](python/)
- [TypeScript / JavaScript](javascript/)
- [Go](go/)
- [Java](java/)
- [.NET](dotnet/)

## Deploy Sigil

Choose your deployment target:

- [Grafana Cloud](grafana-cloud/) — the fastest way to get started with a managed deployment.
- [Kubernetes](kubernetes/) — self-host Sigil using the Helm chart.
